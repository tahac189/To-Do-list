# ✅ To-Do List — DecodeLabs Project 1

A command-line To-Do List application built in Python as part of the **DecodeLabs Junior Python Developer Internship**.

## 🚀 About the Project
This project demonstrates core Python concepts including:
- Lists and the `.append()` method
- Functions and modular code structure
- `enumerate()` for professional looping
- The IPO model: Input → Process → Output
- Entry point pattern with `if __name__ == "__main__"`

## 🛠️ How to Run

```bash
python todo_list.py
```

## 💻 Features
- Add tasks to your list
- View all tasks with numbered display
- Handles empty list gracefully
- Input validation for invalid choices
